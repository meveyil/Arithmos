# 🪙 Arithmos

**Arithmos** is a modern, high-performance, and privacy-centric personal finance tracker. Built with a "local-first" philosophy, it combines the flexibility of web technologies with the power of a native desktop application.

> **Privacy First:** All your financial data stays on your machine. Arithmos works entirely offline and never sends your sensitive information to any external servers.

---

## ✨ Key Features

* **Glassmorphism UI:** A sleek, premium interface featuring an Apple-inspired aesthetic with native-like transparency.
* **GPU Optimized:** Silky-smooth animations and transitions powered by hardware acceleration.
* **Zero-Internet Dependency:** Full functionality without any cloud connection, ensuring your data remains yours.
* **Interactive Analytics:** Visualized spending habits through dynamic and responsive charts.
* **Custom Mechanical Feel:** Interface elements designed with the same attention to detail as a custom keyboard enthusiast's setup.

---

## 🛠 Tech Stack

* **Core:** [Next.js 15](https://nextjs.org/) & [React 19](https://react.dev/)
* **Desktop Environment:** [Electron](https://www.electronjs.org/)
* **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Charts:** [Recharts](https://recharts.org/)
* **Icons:** [Lucide React](https://lucide.dev/)
